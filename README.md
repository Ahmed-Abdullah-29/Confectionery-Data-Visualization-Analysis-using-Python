# Confectionery-Data-Visualization-Analysis-using-Python
A data visualization project analyzing confectionery items dataset to uncover patterns in product types, popularity, and characteristics. Uses Python, Pandas, Matplotlib, and Seaborn to create insightful charts and support data-driven analysis.

## Project Overview

The goal of this project is to explore confectionery items and understand:
- Product distribution and characteristics  
- Popular categories of sweets  
- Relationships between different features  
- Trends in confectionery data  

## Dataset Description

The dataset includes information about various confectionery items such as:
- Product Name  
- Category / Type  
- Ingredients / Attributes  
- Popularity indicators  
- Other descriptive features
  
## Data Preprocessing

Before visualization, the dataset was cleaned by:
- Handling missing values  
- Formatting inconsistent data  
- Converting variables into usable formats  
- Removing duplicates (if present)  
## 📊 Analysis & Visualizations

The notebook includes multiple visual insights:

### 1. Category Distribution
- Bar charts showing the number of products in each category  

### 2. Feature Relationships
- Scatter plots to identify relationships between variables  

### 3. Distribution Analysis
- Histograms to understand data spread  

### 4. Comparative Analysis
- Comparison between different confectionery types
